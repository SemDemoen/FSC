Password Hash Attacks

This page describes common attacks on password hashes such as brute-force, dictionary attacks, rainbow tables, and GPU-accelerated cracking using tools like Hashcat.

Brute Force Attacks

Brute force attempts every possible character combination until a matching hash is found. Strong hashing algorithms and long passwords reduce feasibility.

Dictionary Attacks

Attackers use wordlists containing common passwords or leaked password databases to quickly test likely matches.

Rainbow Table Attacks

Hashcat

Hashcat is a GPU-accelerated password cracking tool capable of trying billions of hashes per second. It supports dictionary, hybrid, brute force, and rule-based attacks.

Salting

Adding a unique, random salt to each password prevents rainbow table attacks and makes cracking much harder, as each salted hash must be cracked individually.
